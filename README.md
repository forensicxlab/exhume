# The Exhume ToolKit
The Exhume ToolKit umbrella repository containing all of the exhume toolkit framework modules for digital forensics. The purpose of this repository is mainely for developpement and from source compilation.

## Getting started

```bash
git clone --recurse-submodules git@github.com:forensicxlab/exhume.git
cargo build
# Example run of the exhume_body module
cargo run --bin exhume_body -- --help
```
